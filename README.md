Protect the Social Network 🛡️

📋 Requirement Fitur:
Screen 1 (Login): Email & Password + Tombol "Daftar Disini".

Screen 2 (Register): Nama, Email, Phone, Pass, Confirm Pass.

Screen 3 (Home): Welcome Message dengan nama user.

🛡️ Security Logic:
Validasi Email (RegEx).
Validasi Phone (Cuma angka & min. 10 digit).
Match Check: Password & Confirm Password harus sama!
Handle Keyboard agar tidak nutupin tombol "Submit".
